{# ========================================================================== reg_comment.render() ========================================================================== Description: Creates an Regulations.gov comment form when given: value: An object used to customize the markup. value.document_id: Federal Register document ID number to which the comment should be submitted. Should follow this format: `CFPB-FRDOC_####-####`. value.id: (optional) Sets the id attribute in the form's markup. If not set, the form will be assigned a base id of `o-reg-comment_` with a random number appended. ========================================================================== #}

If you would like to include an attachment with your response, please submit your comment at Regulations.gov .

Sensitive personal information, such as account numbers or Social Security numbers, should not be included. Your comments will become part of the public record and posted to Regulations.gov . Comments generally will not be edited to remove any identifying or contact information.
